# Architecture

## Routing

- Next.js App Router
- route groups for dashboards
- server-first architecture

## State

- Redux Toolkit
- local component state for UI

## UI

- shadcn/ui
- TailwindCSS

## Testing

- cypress

## Data Fetching

- server actions preferred
- client fetching only for interactive flows

## Folder Structure

- app/ routes
- features/ business logic
- components/ reusable UI
